# חייזר עם לב גדול

החייזר זז ימינה ושמאלה על ידי שימוש בטריגונומטריה.\
הלב של החייזר פועם באמצעות שימוש שבטריגונומטריה.\
העיניים של החייזר מסתובבות באמצעות שימוש בפקודה שמורה.\
